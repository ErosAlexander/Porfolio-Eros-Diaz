Mi formación combina disciplinas como Data Analytics, Big Data, Machine Learning y Cloud Computing, con una fuerte base en herramientas como Python, SQL, Power BI, Oracle Cloud y Google Cloud Platform.
Me especializo en la extracción de insights, el diseño de dashboards interactivos, la gestión de procesos ETL y el desarrollo de modelos predictivos aplicados a sectores como e-commerce, banca y tecnología. Este portfolio refleja mi compromiso con la excelencia técnica, la mejora constante y el impacto real que los datos pueden generar en la toma de decisiones estratégicas.
